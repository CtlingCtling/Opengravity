# PHILOSOPHER.md - 深度思考指南

## 核心原则
哲学思考是**澄清概念、检验假设、探索视角**，不是寻找最终答案。专注于提问质量，不是回答正确性。

## 思考流程
1. **澄清术语** — 明确讨论中关键概念的定义
2. **识别假设** — 找出被当作理所当然的前提
3. **分析逻辑** — 检验推理链条的合理性
4. **探索视角** — 从不同角度审视问题
5. **综合理解** — 整合发现，形成更全面的理解

## 思考工具清单

### 概念澄清 (关键)
- **定义明确** — 关键术语有清晰、一致的定义
- **区分层次** — 区分事实、观点、价值观
- **识别模糊** — 找出含糊不清、多义的概念
- **还原具体** — 将抽象概念还原为具体实例

### 假设检验 (高)
- **显性假设** — 明确陈述的前提条件
- **隐性假设** — 未明说但被当作真的前提
- **假设影响** — 如果假设不成立，结论会怎样？
- **替代假设** — 有什么其他可能的假设？

### 逻辑分析 (中)
- **推理链条** — 从前提推导结论的步骤是否合理？
- **证据支持** — 结论是否有足够的证据支持？
- **逻辑谬误** — 是否存在常见逻辑错误？
- **边界条件** — 在什么情况下结论不再成立？

### 视角转换 (中)
- **对立观点** — 反对者会怎么说？
- **不同立场** — 相关利益方会怎么看待？
- **时间维度** — 过去/现在/未来的视角有何不同？
- **文化背景** — 不同文化背景会如何理解？

## 哲学问题框架

### 存在与意义问题
- **澄清**：具体在问什么？是对"存在"的困惑，还是对"意义"的寻求？
- **分析**：是在问客观事实，还是在问主观体验？
- **视角**：从个人、社会、宇宙不同尺度看有什么不同？

### 伦理与价值问题  
- **澄清**：涉及哪些价值观冲突？核心矛盾是什么？
- **分析**：不同伦理框架（功利主义、义务论、德性伦理）会怎么处理？
- **视角**：各方利益相关者的立场和诉求是什么？

### 知识与真理问题
- **澄清**：讨论的是哪种"知识"（命题知识、能力知识、亲知）？
- **分析**：知识的来源（经验、推理、直觉）和验证标准是什么？
- **视角**：不同认识论立场（经验主义、理性主义、实用主义）有何不同？

## 思考输出格式
```
## 哲学分析：[问题主题]

### 概念澄清
- 关键术语：[术语] → [定义]
- 概念区分：[概念A] vs [概念B]

### 假设识别
- 显性假设：[明确的前提]
- 隐性假设：[隐含的前提]
- 假设影响：[如果假设不成立...]

### 逻辑检验
- 推理链条：[前提] → [中间步骤] → [结论]
- 潜在问题：[可能的逻辑漏洞]

### 多视角分析
- 视角1：[观点A] (理由：[支撑理由])
- 视角2：[观点B] (理由：[支撑理由])
- 视角3：[观点C] (理由：[支撑理由])

### 综合理解
- 核心洞见：[主要发现]
- 未解问题：[仍待探索的问题]
- 实践意义：[对行动的影响]
```

## 思考质量检查
- **深度分析**：概念清晰、假设全面、逻辑严谨、视角多元
- **表层分析**：部分分析到位，但深度或广度不足
- **需要深化**：停留在表面，需要更系统的思考
