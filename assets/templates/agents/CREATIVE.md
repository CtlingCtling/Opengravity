# CREATIVE.md - 创意生成指南

## 核心原则
创意是**组合现有元素**产生新价值，不是凭空创造。专注于连接、重组、转换，而不是从零开始。

## 创意流程
1. **收集素材** — 广泛收集相关信息、想法、灵感
2. **发散思考** — 不加评判地生成大量可能性
3. **寻找连接** — 发现不同素材间的关联和模式
4. **组合创新** — 将不同元素组合成新概念
5. **评估优化** — 筛选最有价值的创意，进一步完善

## 创意技术清单

### 头脑风暴 (关键)
- **数量优先** — 先追求数量，不评判质量
- **推迟评判** — 生成阶段禁止批评和筛选
- **鼓励疯狂** — 欢迎看似不切实际的想法
- **组合改进** — 在他人的想法基础上改进

### 连接方法 (高)
- **类比思考** — "这像什么？"寻找相似领域
- **强制关联** — 随机选择两个不相关元素强制连接
- **反转假设** — "如果相反的情况成立会怎样？"
- **极端缩放** — "放大100倍或缩小100倍会怎样？"

### 突破障碍 (中)
- **改变环境** — 换地方、换工具、换媒介
- **限制条件** — 给自己设定限制（时间、材料、格式）
- **角色扮演** — 以不同角色的视角思考
- **随机输入** — 使用随机词、图片、概念作为起点

### 评估标准 (中)
- **新颖性** — 想法是否新鲜、独特？
- **可行性** — 是否有实现的可能？
- **价值性** — 是否能解决问题或创造价值？
- **共鸣度** — 是否能引起他人兴趣或共鸣？

## 创意输出格式
```
## 创意概念：[概念名称]

### 核心想法
- **是什么**：[简要描述]
- **解决什么问题**：[问题陈述]
- **独特之处**：[创新点]

### 灵感来源
- 元素1：[来源描述]
- 元素2：[来源描述]
- 连接方式：[如何组合]

### 实现路径
1. [第一步：探索/验证]
2. [第二步：原型/测试]
3. [第三步：完善/推广]

### 潜在挑战
- [可能的问题1] → [应对思路]
- [可能的问题2] → [应对思路]
```

## 创意质量检查
- **优秀创意**：新颖、可行、有价值、有共鸣
- **良好创意**：满足2-3个标准，需要进一步完善
- **需重新构思**：缺乏独特性或价值
